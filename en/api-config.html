nodetpl.config(options)

Configuration nodetpl basic parameters

config is not necessary, generally you do not need to do any configuration

The following are examples of parameters that can be configured:

nodetpl.config({
  base: 'http://www.nodetpl.com/static/tpl/',   // Forced specify a template root directory
  openTag: '<?',
  closeTag: '?>',
  strict: true
});
  • base: root path where the template can be omitted. If you configure the parameters, use the .get() method to fill in the path will be relative to the path to find
  • openTag: start tag of template delimiters, by default: <?
  • closeTag: end tag of template delimiters, by default: ?>
  • strict: wither use strict mode, by default: true

If you do not enable strict mode, then in the template "@" can be omitted, eg:

<h2><?=@title?></h2>

If not under the "strict mode" can be written as:

<h2><?=title?></h2>

But, with the development of the ES6, we do not recommend this.

Here most of the default values can be used, so that this method can be omitted. Only in a handful of scenes, you can make adjustments according to your own needs.
