nodetpl methods

  • nodetpl.config: basic configuration, not necessary
  • nodetpl.get: get a template or precompiled js file
  • nodetpl.render: rendering a template
  • nodetpl.exec: js code execution inside the template, less used

NodeTplClass method

  • tpl.render: rendering the template

tpl.render([data]);

Use data, direct rendering template code. tpl is a NodeTplClass instance, this method can only be used in AMD / CMD / CommonJs / ES6 or "suspend rendering" scene.

tpl.render is a synchronous method, which gives the result directly rendered, and essentially different from nodetpl.render.

var tpl = require('./tpls/1');
var html = tpl.render(data);
// console.log(html);

Use AMD/CMD/CommonJs/ES6 + precompiled develop the biggest advantage is that, you only need to include the template executable files with require, without further reference nodetpl core files, and across platforms in any project, which is the most respected development mode by author.
