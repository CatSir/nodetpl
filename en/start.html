Quick Start

Import file

Download and unzip the nodetpl package.

General reference, link the nodetpl file in header tag (package: ./lib/nodetpl.min.js):

<script src="./js/nodetpl.min.js"></script>

AMD/CMD/CommonJs/ES6：

var nodetpl = require('nodetpl');

Writing Templates

Easiest way, you can use a type="text/template" marked script tag to store the template code:

<script id="myfavor" type="text/template">
<h1><?=@title?></h1>
<ul>
  <?for(var i=0; i<@favor.length; i++){?>
    <li><?=i?>：<?=@favor[i]?></li>
  <?}?>
</ul>
</script>

Perform rendering

var data = {
  "title": "Favor",
  "favor": [
    "Football",
    "Basketball",
    "Table tennis",
    "Glass ball"
  ]
};
var content = document.getElementById('myfavor').innerHTML;
nodetpl.render(content, data, function(d){
  console.log(d);
});

Result

<h1>Favor</h1>
<ul>
  <li>0: Football</li>
  <li>1: Basketball</li>
  <li>2: Table tennis</li>
  <li>3: Glass ball</li>
</ul>
