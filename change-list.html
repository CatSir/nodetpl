升级日志

2.1 to 2.2
  • 编译工具做重大升级，不再要求输入 -o 属性，支持文件监听，支持指定模板后缀
2.0 to 2.1
  • 关键字 “@” 可以省略，在将来的版本中将要废弃
1.x to 2.0
  • 支持通过 config 灵活配置引擎开关
  • 内置 _fixcss 方法即将废弃，使用 1.x 版本编译的模板需要重新编译
  • 正式支持 CMD 模块化开发
  • 多模板引用由 “require” 关键字改为 “include”，原 “require” 关键字废弃
  • 增加关键字 ROOT，代表原生 DOM 节点
  • 默认 $ROOT 指向为 jQuery 对象（选择器 $("#~") ），如需使用其它类库，请手动 config 设置 
    nodetpl.config({
  vars: {
    root: '...'
  }
});
  • render() 由异步方法改为同步
