快速上手

引入文件

下载 nodetpl并解压 nodetpl 包文件

在页面 head 标签内，引入 nodetpl 客户端文件（压缩包中的 /dist/nodetpl.client.min.js）：

<script src="./static/js/nodetpl.client.min.js"></script>

编写模板

最简单的，可以使用一个 type="text/template" 的 script 标签存放模板：

<script id="testid" type="text/template">
<h1><?=title?></h1>
<ul>
  <?for(var i=0; i<favor.length; i++){?>
    <li><?=i?>：<?=favor[i]?></li>
  <?}?>
</ul>
</script>

执行渲染

var data = {
  "title": "个人爱好",
  "favor": ["足球", "篮球", "乒乓球", "琉璃球"]
};
var content = document.getElementById('testid').innerHTML;
var d = nodetpl.render(content, data);
console.log(d);

运行结果

<h1>个人爱好</h1>
<ul>
  <li>0：足球</li>
  <li>1：篮球</li>
  <li>2：乒乓球</li>
  <li>3：琉璃球</li>
</ul>
