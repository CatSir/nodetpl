nodetpl 方法

nodetpl.config(options)

配置 nodetpl 基本参数

nodetpl.config({
  base: 'http://www.nodetpl.com/static/js/tpl/',   // 模板目录
  vars: {
    root: '$("#~")'
  },
  openTag: '<?',
  closeTag: '?>',
  cache: true
});
  • base：模板所在的根路径，可以省略
  • vars：变量相关信息
    • root：模板中 JavaScript 代码里的 $ROOT 所代表的对象。默认为 DOM 节点，即 document.getElementById("~")。如果你使用的是 jQuery 库，可以改为：$("#~")
  • openTag：模板定界符开始标签，默认为：<?
  • closeTag：模板定界符结束标签，默认为：?>
  • cache：是否开启缓存，默认开启

这里大部分是可以采用默认值的，通常状况下，可以根据所使用的类库，调整 vars.root 的指向。

如果你使用的是 jQuery，那么设置 $("#~") 之后，$ROOT 便是模板代码中的 <div id="$ROOT"> 根节点。

nodetpl.get(url, [data, ] callback)

加载指定的模板可执行文件（编译方式参见预编译），并用 data 渲染，渲染后的结果返回给 callback。

  • url：模板编译后的文件
  • data：传入模板的数据
  • callback：模板编译后，执行的回调函数，回调函数会将编译后的 HTML 作为参数传入
var data = {
  title: '个人爱好',
  favor: ['足球', '篮球', '乒乓球', '琉璃球']
};
nodetpl.get('http://www.nodetpl.com/tpls/demo/test.js', data, function(d){
  alert(d);
});

nodetpl.render(tplcontent[, data]);

使用数据 data，渲染模板代码 tplcontent，此方法不包含预编译环节，通常用在人为拼接或者 <script type="text/template"... 的场景。

var data = {
  title: '个人爱好',
  favor: ['足球', '篮球', '乒乓球', '琉璃球']
};
var content = document.getElementById('testid').innerHTML;
var html = nodetpl.render(content, data);
alert(html);
