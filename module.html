CMD（seaJs）

seaJs 基本配置

配置 seaJs 基本参数，增加 nodetpl 的 alias 项：

seajs.config({
  alias: {
    'nodetpl': 'lib/nodetpl.client.min'
  }
});

通过 require 将 nodetpl 模块引入：

var nodetpl = require('nodetpl');
nodetpl.get('...');
