nodetpl 方法

  • nodetpl.config：基本配置，非必须
  • nodetpl.get：获取某个模板或编译后的模板文件
  • nodetpl.render：渲染某个模板
  • nodetpl.exec：执行模板里面的 js 代码，较少使用

NodeTplClass 方法

  • tpl.render：渲染模板

tpl.render([data]);

使用数据 data，直接渲染模板，其中，tpl 是一个 NodeTplClass 实例，该方法只能用在 AMD/CMD/CommonJs/ES6 或“暂缓渲染”的场景中。

tpl.render 是一个同步方法，该方法会直接给出渲染后的结果，它和 nodetpl.render 有着本质的区别。

var tpl = require('./tpls/1');
var html = tpl.render(data);
// console.log(html);

使用 AMD/CMD/CommonJs/ES6 + 预编译 开发最大的优势在于，你只需要通过 require 引入模板可执行文件，无需再引用 nodetpl 核心文件，可以跨平台运行在任何项目中，这也是作者最为推崇的一种开发方式。
