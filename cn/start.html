快速上手

引入文件

下载 nodetpl并解压。

常规引用，在页面 head 标签内，引入 nodetpl（压缩包中的 ./lib/nodetpl.min.js）：

<script src="./js/nodetpl.min.js"></script>

AMD/CMD/CommonJs/ES6：

var nodetpl = require('nodetpl');

编写模板

最简单的，可以使用一个 type="text/template" 的 script 标签存放模板：

<script id="myfavor" type="text/template">
<h1><?=@title?></h1>
<ul>
  <?for(var i=0; i<@favor.length; i++){?>
    <li><?=i?>：<?=@favor[i]?></li>
  <?}?>
</ul>
</script>

执行渲染

var data = {
  "title": "个人爱好",
  "favor": ["足球", "篮球", "乒乓球", "琉璃球"]
};
var content = document.getElementById('myfavor').innerHTML;
nodetpl.render(content, data, function(d){
  console.log(d);
});

运行结果

<h1>个人爱好</h1>
<ul>
  <li>0：足球</li>
  <li>1：篮球</li>
  <li>2：乒乓球</li>
  <li>3：琉璃球</li>
</ul>
