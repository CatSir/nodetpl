nodetpl 方法

  • nodetpl.config：基本配置，非必须
  • nodetpl.get：获取某个模板或编译后的模板文件
  • nodetpl.render：渲染某个模板
  • nodetpl.exec：执行模板里面的 js 代码，较少使用

NodeTplClass 方法

  • tpl.render：渲染模板

nodetpl.render(tplcontent[, data][, callback]);

使用数据 data，直接渲染模板代码 tplcontent。

此方法不包含预编译环节，编译过程由浏览器执行，通常用在人为拼接或者 <script type="text/template"... 的场景。

nodetpl.render 是一个异步方法，该方法会在回调函数里给出渲染后的结果，它和 tpl.render 有着本质的区别。

HTML 部分：

<script id="favor-tpl" type="text/template">
<h1><?=@title?></h1>
<ul>
  <?for(var i=0; i<@favor.length; i++){?>
    <li><?=i?>：<?=@favor[i]?></li>
  <?}?>
</ul>
</script>

JS 部分：

var data = {
  title: '个人爱好',
  favor: ['足球', '篮球', '乒乓球', '琉璃球']
};
var content = document.getElementById('favor-tpl').innerHTML;
nodetpl.render(content, data, function(d){
  console.log(d);
});

或者，不写在 HTML 中，直接在 js 中做字符串拼接亦可：

var template = '\
<script id="favor-tpl" type="text/template">\
<h1><?=@title?></h1>\
<ul>\
  <?for(var i=0; i<@favor.length; i++){?>\
    <li><?=i?>：<?=@favor[i]?></li>\
  <?}?>\
</ul>\
</script>';
