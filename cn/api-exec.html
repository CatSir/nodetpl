nodetpl 方法

  • nodetpl.config：基本配置，非必须
  • nodetpl.get：获取某个模板或编译后的模板文件
  • nodetpl.render：渲染某个模板
  • nodetpl.exec：执行模板里面的 js 代码，较少使用

NodeTplClass 方法

  • tpl.render：渲染模板

nodetpl.exec(content);

执行 html 中的 JavaScript，该方法在极少数场景下使用（例如原生 innerHTML），因为大多数 js 库已包含自动执行的方法。

例如下面一段 HTML，通过常规 innerHTML 添加到某个 DOM 中时，JavaScript 不会执行，在添加完之后手动运行一下 exec 方法即可。

<h1>hello</h1>
<script>
  alert(0);
</script>
nodetpl.exec(content);
